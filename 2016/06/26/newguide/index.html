综述

效率第一！

VPS系统环境：本教程只适用于Ubuntu 建议选择14.04 x86更节省硬盘空间！
所需SSH软件：点我下载PuTTY
所需FTP软件：点我下载Filefilla

注意：在PuTTY中鼠标左键选中的内容会自动被复制，单击鼠标右键是黏贴(在PuTTY中不要使用ctrl+c/ctrl+v复制粘贴！)

在linux中强制终止程序:CTRL+c

开通vps并安装好系统，记住【IP地址】、【ssh端口】(默认是22)和【root密码】

使用PuTTY
HostName(or IP address):【IP地址】
Port:22
(点save保存后open,选择YES)
login as:root
password:【root密码】
(粘贴你的root密码，输入后无任何显示，直接回车)

使用FTP软件FileZilla
主机host：sftp://【IP地址】
用户名：root
密码：【root密码】
端口：22
快速连接(Q)旁边小三角为账号记录，下次直接点击已保存的账号登陆

screen用法：
为了防止关掉putty、网络断掉等造成安装失败、下载上传任务中断，务必使用screen
常用命令：
创建会话：screen -S download 创建了一个名叫download的会话，请在会话中进行配置、下载、上传等操作
隐藏会话：键盘按下Ctrl+a松开后按d（后台程序在断开putty的情况下也会继续运行）
列举会话：screen -ls
回到会话：screen -r 会话/代码比如例子:screen -r download或者screen -r 会话代码回车，就会看到之前继续的进程
杀死当前会话：在会话中按下Ctrl+a松开后按k

下载方法

首先安装youtube-dl，you-get：
pip3 install youtube-dl you-get
进入我们存放文件的目录/var/www/html
cd /var/www/html
主要介绍下载youtube的方法，其他网站(如naver、youku等请点击更多介绍
使用youtube-dl，可以查找本站教程或者官网：
youtube-dl -f bestvideo[ext=mp4]+bestaudio[ext=m4a] -o '%(title)s' --all-subs -i URL1 URL2 URL3

使用you-get详细中文介绍：
you-get --itag=266 URL1 URL2 URL3
–itag=266：266是选择视频格式代码，最好在下载前先使用you-get -i URL查看可下载的格式

由于目前you-get的格式选择功能比较弱，所以大量下载还是推荐youtube-dl

youtube-dl和you-get都支持 -o选项 加入绝对路径，这样方便在任何工作目录内都可以直接使用语句下载:
youtube-dl加入:-o "/var/www/html/新创建目录1/%(title)s"
you-get加入:-o "/var/www/html/新创建目录1"

另：使用wget更多介绍下载远程服务器上的文件：
下载单个文件:wget --restrict-file-names=nocontrol 'URL'

另：下载整个目录(包括其子目录):wget -c -r -np -L -k --restrict-file-names=nocontrol URL

选项一 bpcs_uploader上传百度云

依次执行：
cd ~
unzip master
mv oott123-bpcs_uploader-3a75324 baidupan注意中间一长串内容为上一步unzip master中对应的文件夹名称,不要弄错
cd baidupan
获取运行权限：
chmod +x bpcs_uploader.php
授权：
./bpcs_uploader.php quickinit

注意：复制图中红圈位置标出的授权码，每次授权都是不同的，图中为rqxtjqzk
浏览器打开网址：https://openapi.baidu.com/device
输入你自己显示的授权码，网页显示”您已成功为bpcs_uploader授权！”之后回到putty，回车继续
授权成功显示：Access Granted. Your Storage Status: 1791.33G/2056.00G (87.13%)Enjoy!

如果提示php错误，首先确定php的绝对路径：witch php然后vi bpcs_uploader.php
确认第一行中红线部分是否和刚才显示的路径一致(vi的用法在后文脚本介绍中)

授权有效期为一个月，更换百度盘账号或者重新授权方法：cd /root/baidupan之后./bpcs_uploader.php quickinit
项目主页介绍

由于api权限原因，bpcs_uploader只能存取百度盘->我的应用数据->bpcs_uploader目录下面的文件和目录，注意文件名用双引号。

用bpcs_uploader来上传目录，请使用后文介绍的脚本

选择二 PCS上传百度云

PCS项目主页介绍的非常详细，推荐看一看，以下简单地介绍我们所需要的命令！
登陆：
pcs login
输入账号和密码

如果提示输入验证码：
浏览器打开：http://【IP地址】/captch.gif
查看并输入验证码

上传前请先确认百度盘内是否有此目录，如果无请先使用pcs mkdir 目录名称在百度上创建一个目录
比如在百度上创建一个upload文件夹：pcs mkdir upload

(PCS不同于bpcs，它不受api限制！)

为了保持vps和百度云盘目录同步，并配合bpcs_uploader，我们使用PCS时，可以上传到百度盘的/apps/bpcs_uploader目录(/apps等于我的应用数据)
如果为了配合bypy就将上面命令中的bpcs_uploader改成bypy

上传目录：
pcs synch -u -r -c /var/www/html /upload
解释：
-u:只上传需要上传的文件，等价于备份
-r:递归上传，使用此选项会上传/var/www/html目录包括其子目录内所有文件
-c:显示上传结果，防止漏上传或下载
/var/www/html：vps上的/var/www/html目录
/upload：上传到百度盘–/upload内

将上一句中的synch换成compare就可以比对vps里和/upload内的文件了

上传文件：
pcs u <local file> <remote file>
解释：
local file：格式为本地路径/文件名.后缀 注意文件名的引号
remote file：格式为百度盘路径/文件名.后缀 注意文件名的引号

查看百度盘容量：pcs quota
退出账号：pcs logout

脚本介绍

vi的用法,也可以使用vim或gedit
熟悉基本的打开文件，修改文件，保存，退出等用法
打开某xxx文件:vi xxx.sh 切换到插入模式：键盘的i或者INS
保存并退出：ESC之后ZZ，退出不保存：ESC之后依次输入:q!

脚本运行方式：
在root目录下./xxx.sh xxx.sh为脚本文件名

bpcs_uploader上传脚本：
bdup.sh用来上传/var/www/html目录内所有文件，但是不能上传它的子目录
编辑脚本：vi /root/bdup.sh

updir.sh用来上传/var/www/html子目录的文件，比如输入新创建目录1或者新创建目录1/新创建目录1.1
编辑脚本：vi /root/updir.sh

PCS上传脚本：
pdup.sh用来上传/var/www/html目录内包括子目录内所有文件，本脚本中上传到和bpcs-uploader同一个云端目录，如果需要下载到upload文件夹请修改remotedir="/upload"
编辑脚本：vi /root/pdup.sh

delall.sh用来删除html下所有文件(包括子目录，慎用！)
编辑脚本：vi /root/delall.sh

当然你也可以自己写上传单文件、下载等，方便使用！
